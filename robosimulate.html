ROS based simulation tool for serial robots

ROS based simulation software for manipulation of 6 DOF robots. The software will serve as a generic simulation tool for robot manipulation by choosing Inverse kinematics solutions, path planning, adding collision objects, collision avoidance etc. This tool will come in handy for doctors in surgical environments where they can pre visualize the overall workflow of the robot movement and path. The system is designed in a more dynamic way, where user can choose different functionalities with their needs. (Example : User can explicitly mention about the path in which robot has to move with defined constraints and defined orientation). The prototype is developed using Universal Robot 5e. Later can be integrated to other commercial robots
